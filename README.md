## Overview

FAST PIZZA is a web application built with React that allows users to quickly and easily place pizza orders. It provides a user-friendly interface for selecting pizzas, customizing toppings, and completing the order process.

## Features

- **Intuitive Interface:** Easy-to-use design for a seamless ordering experience.
- **Pizza Selection:** Browse and select from a variety of pizza options.
- **Customization:** Customize your pizza with different toppings and sizes.
- **Order Management:** Manage your current order and review past orders.
- **Responsive Design:** Works perfectly on desktops, tablets, and mobile devices.

## Installation

1.  Clone the repository:

    ```bash
    git clone https://github.com/iwanehu/fast-pizza.git
    ```

2.  Navigate to the project directory:

    ```bash
    cd fast-pizza
    ```

3.  Install the dependencies:

    ```bash
    npm install
    ```

## Usage

1.  Start the development server:

    ```bash
    npm run dev
    ```

2.  Open your browser and navigate to the address provided in the console (usually `http://localhost:5173`).
